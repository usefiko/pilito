# 🎯 Intent Keywords System - راهنمای کامل

## 📌 این سیستم چیه؟

**Intent Keywords** یک سیستم تشخیص هوشمند منظور کاربر است که به AI کمک می‌کند بفهمد:
- کاربر **چی می‌خواد**؟
- موضوع سوالش **چیه**؟
- باید از **کدوم منبع دانش** جواب بده؟

---

## 🤔 مثال ساده:

وقتی کاربر می‌پرسه:
> "قیمت این محصول چقدره?"

سیستم باید بفهمه که:
- ✅ **Intent = Pricing** (موضوع اصلی: قیمت)
- ✅ باید بره تو **Products** بگرده (نه FAQ یا Manual)
- ✅ کلمه کلیدی: "قیمت" و "چقدر"

---

## 🎯 چرا مهمه؟

### ❌ **بدون Intent Keywords:**
```
کاربر: "قیمت چقدره؟"
       ↓
AI: بگرد تو همه جا! (FAQ + Manual + Products + Website)
       ↓
نتیجه: 
  - زمان: 3-5 ثانیه 😴
  - هزینه: 2000 توکن 💸
  - دقت: متوسط 😐
```

### ✅ **با Intent Keywords:**
```
کاربر: "قیمت چقدره؟"
       ↓
Intent Detection: "Pricing" شناسایی شد!
       ↓
AI: فقط تو Products بگرد!
       ↓
نتیجه:
  - زمان: 0.8 ثانیه ⚡
  - هزینه: 600 توکن 💰
  - دقت: بالا ✨
```

---

## 📊 پنج Intent اصلی:

### 1️⃣ **Pricing & Plans** (قیمت و تعرفه)
**زمانی که کاربر می‌پرسه:**
- قیمت چقدره؟
- چند تومنه؟
- تخفیف داری؟
- هزینه‌اش چیه؟
- اشتراک چقدره؟

**منبع دانش:** Products, FAQ

---

### 2️⃣ **Product Info** (اطلاعات محصول)
**زمانی که کاربر می‌پرسه:**
- چه محصولاتی دارید؟
- ویژگی‌های این محصول چیه؟
- مشخصات فنی؟
- این محصول چیکار می‌کنه؟
- تفاوتش با محصول دیگه چیه؟

**منبع دانش:** Products, Website

---

### 3️⃣ **How-to & Tutorial** (آموزش و راهنما)
**زمانی که کاربر می‌پرسه:**
- چطور استفاده کنم؟
- راهنمای نصب؟
- چگونه تنظیم کنم؟
- آموزش کار با این؟
- مراحل راه‌اندازی؟

**منبع دانش:** Manual, Website, FAQ

---

### 4️⃣ **Contact & Support** (تماس و پشتیبانی)
**زمانی که کاربر می‌پرسه:**
- چطور باهاتون تماس بگیرم؟
- شماره تماس؟
- آدرس دفتر؟
- پشتیبانی کجاست؟
- ایمیل شما؟

**منبع دانش:** FAQ, Manual

---

### 5️⃣ **General Question** (سوالات عمومی)
**زمانی که کاربر می‌پرسه:**
- سلام
- شما کی هستید؟
- کمک
- درباره شرکت شما
- هر سوال کلی دیگه

**منبع دانش:** FAQ, Manual, Website, Products

---

## 🌍 زبان‌های پشتیبانی شده:

1. **فارسی (fa)** - اصلی
2. **English (en)** - بین‌المللی
3. **العربية (ar)** - عربی
4. **Türkçe (tr)** - ترکی

---

## 📝 ساختار یک Keyword:

```json
{
  "intent": "pricing",           // نوع موضوع
  "language": "fa",              // زبان
  "keyword": "قیمت",             // کلمه کلیدی
  "weight": 2.0,                 // اهمیت (0.1-3.0)
  "user": null,                  // null = برای همه کاربرها
  "is_active": true              // فعال/غیرفعال
}
```

---

## ⚖️ سیستم وزن‌دهی (Weight):

| وزن | معنی | مثال |
|-----|------|------|
| **3.0** | فوق‌العاده مهم | "قیمت دقیق" |
| **2.0-2.5** | خیلی مهم | "قیمت"، "چقدر"، "محصول" |
| **1.5** | مهم | "هزینه"، "تعرفه"، "ویژگی" |
| **1.0** | معمولی | "اطلاعات"، "جزئیات" |
| **0.5** | کم اهمیت | کلمات کمکی |

---

## 🎯 چه Keywords هایی باید تولید بشه؟

### برای **Pricing**:
- کلماتی که مستقیماً **قیمت** رو می‌پرسن
- کلماتی که درباره **هزینه** هستن
- کلماتی که درباره **تخفیف** هستن
- کلماتی که درباره **پرداخت** هستن

### برای **Product**:
- کلماتی که **محصول** رو می‌پرسن
- کلماتی که درباره **ویژگی** هستن
- کلماتی که درباره **مشخصات** هستن
- کلماتی که **مقایسه** می‌کنن

### برای **Howto**:
- کلماتی که **چطور** می‌پرسن
- کلماتی که درباره **آموزش** هستن
- کلماتی که **راهنما** می‌خوان
- کلماتی که **نصب** رو می‌پرسن

### برای **Contact**:
- کلماتی که **تماس** می‌خوان
- کلماتی که **آدرس** می‌پرسن
- کلماتی که **پشتیبانی** می‌خوان
- کلماتی که **شماره** می‌پرسن

### برای **General**:
- **سلام**، **خداحافظ**
- **کمک**، **راهنما**
- **درباره شما**
- **کی هستید**

## ✅ نکات مهم برای تولید:

1. **تنوع داشته باش:**
   - قیمت، هزینه، چقدر، چند تومن، تعرفه (همه معنی مشابه ولی کلمات متفاوت)

2. **فرهنگ محلی:**
   - فارسی: "چند تومنه؟"
   - عربی: "كم السعر؟"
   - ترکی: "Fiyatı ne kadar?"

3. **کلمات ترکیبی:**
   - "چه قیمتی"
   - "چطور کار می‌کنه"
   - "راهنمای استفاده"

4. **املای متفاوت:**
   - "می‌کنه" و "میکنه"
   - "راهنما" و "راهنمای"

5. **عامیانه هم اضافه کن:**
   - "چنده؟" (به جای "چند است؟")
   - "میخوام" (به جای "می‌خواهم")
