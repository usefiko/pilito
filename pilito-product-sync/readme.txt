سینک خودکار محصولات فروشگاه با پلتفرم پیلیتو برای استفاده از هوش مصنوعی

== Description ==

این پلاگین محصولات فروشگاه WooCommerce شما را به‌صورت خودکار با پلتفرم پیلیتو سینک می‌کند تا بتوانید از قابلیت‌های هوش مصنوعی پیلیتو برای:
- پاسخگویی خودکار به مشتریان
- توصیه هوشمند محصولات
- پشتیبانی 24/7
- تحلیل رفتار مشتریان

استفاده کنید.

= ویژگی‌ها =

* ✅ سینک خودکار محصولات جدید
* ✅ به‌روزرسانی خودکار تغییرات
* ✅ حذف خودکار محصولات پاک شده
* ✅ سبک و بدون فشار به سرور
* ✅ نمایش وضعیت سینک در لیست محصولات
* ✅ تست ساده اتصال
* ✅ لاگ اختیاری برای debugging

= نحوه استفاده =

1. پلاگین را نصب و فعال کنید
2. به داشبورد پیلیتو بروید و یک توکن API ایجاد کنید
3. توکن را در تنظیمات پلاگین وارد کنید
4. اتصال را تست کنید
5. تنظیمات را ذخیره کنید

از این پس تمام تغییرات محصولات به‌صورت خودکار با پیلیتو سینک می‌شود!

== Installation ==

1. فایل zip پلاگین را آپلود کنید
2. پلاگین را فعال کنید
3. به WooCommerce > پیلیتو Sync بروید
4. دستورالعمل‌های نصب را دنبال کنید

== Frequently Asked Questions ==

= آیا نیاز به تنظیمات پیچیده دارد؟ =

خیر! فقط کافیست توکن API را از داشبورد پیلیتو کپی کرده و در تنظیمات پلاگین وارد کنید.

= آیا محصولات موجود هم سینک می‌شوند؟ =

بله، هر محصولی که بعد از فعال‌سازی پلاگین ویرایش شود، سینک خواهد شد.

= آیا بر عملکرد سایت تأثیر می‌گذارد؟ =

خیر، پلاگین بسیار سبک طراحی شده و از ارسال non-blocking استفاده می‌کند.

= با چه پلتفرم‌های فروشگاهی کار می‌کند؟ =

در حال حاضر فقط با WooCommerce، ولی به‌زودی سایر پلتفرم‌ها نیز پشتیبانی خواهند شد.
