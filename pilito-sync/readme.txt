همگام‌سازی خودکار محتوای سایت با پلتفرم پیلیتو برای استفاده از هوش مصنوعی

== Description ==

این پلاگین محتوای سایت شما (محصولات WooCommerce، صفحات و نوشته‌ها) را به‌صورت خودکار با پلتفرم پیلیتو همگام می‌کند تا بتوانید از قابلیت‌های هوش مصنوعی پیلیتو برای:
* پاسخگویی خودکار به مشتریان
* توصیه هوشمند محصولات
* پشتیبانی 24/7
* تحلیل رفتار مشتریان

استفاده کنید.

= ویژگی‌ها =

* ✅ همگام‌سازی خودکار محصولات WooCommerce (اختیاری)
* ✅ همگام‌سازی صفحات و نوشته‌های وردپرس
* ✅ به‌روزرسانی خودکار تغییرات
* ✅ حذف خودکار محتوای پاک شده
* ✅ سبک و بدون فشار به سرور
* ✅ نمایش وضعیت همگام‌سازی
* ✅ تست ساده اتصال
* ✅ رابط کاربری ساده و حرفه‌ای
* ✅ پشتیبانی از زبان‌های فارسی و انگلیسی

= نحوه استفاده =

1. پلاگین را نصب و فعال کنید
2. به داشبورد پیلیتو بروید و یک توکن API ایجاد کنید
3. توکن را در بخش تنظیمات پلاگین وارد کنید
4. اتصال را تست کنید
5. تنظیمات را ذخیره کنید

از این پس تمام تغییرات محتوا به‌صورت خودکار با پیلیتو همگام می‌شود!

== Installation ==

1. فایل zip پلاگین را از WordPress.org دانلود کنید
2. به بخش افزونه‌ها > افزودن > آپلود فایل بروید
3. فایل zip را آپلود و فعال کنید
4. به پیلیتو > تنظیمات بروید
5. توکن API را وارد و ذخیره کنید

== Frequently Asked Questions ==

= آیا نیاز به WooCommerce دارم؟ =

خیر! پلاگین بدون WooCommerce هم کار می‌کند و می‌توانید فقط صفحات و نوشته‌ها را همگام کنید.

= آیا محصولات موجود هم همگام می‌شوند؟ =

بله، هر محصولی که بعد از فعال‌سازی پلاگین ویرایش شود، همگام خواهد شد. همچنین می‌توانید از بخش "محصولات" همه محصولات را به صورت دستی همگام کنید.

= آیا بر عملکرد سایت تأثیر می‌گذارد؟ =

خیر، پلاگین بسیار سبک طراحی شده و از ارسال non-blocking استفاده می‌کند.

= با چه پلتفرم‌های فروشگاهی کار می‌کند؟ =

در حال حاضر فقط با WooCommerce، ولی به‌زودی سایر پلتفرم‌ها نیز پشتیبانی خواهند شد.

= آیا محتوای صفحات و نوشته‌ها هم همگام می‌شود؟ =

بله، می‌توانید از بخش "صفحات و نوشته‌ها" صفحات و نوشته‌های مورد نظر خود را انتخاب و همگام کنید.
